This is the help manual to use Ji's todo v2 command line application.
Here are the possible options that you can use:
    node index.js + (one of the options below):

        add "some todo item"                to add an item to the todo list
        list                                to list the todo items
        remove <position>                   to remove a specific todo item
        update <position>                   to modify an existing todo item
        mark <position> "mark"              to mark a todo item with a specific state. ex: DONE
        drlmark <position>                  to remove the mark added before
        reset                               to clear the todo list
        help                                to display this manual
