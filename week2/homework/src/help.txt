
===This program helps to create a to-do list.===

Usable commands:

list : 'Shows current todo list',

reset : 'Removes all todo items from the list.'

add [todo] : 'Adds a [todo] item to list.'

remove [number] : 'Removes a todo item by depending on its 1-base index [number].'

update [number] [todo]: 'Updates todo number [number] in the list with new [todo]'
