Instructions:
=============


help:
-----
To display this set of instructions:
node index.js help


list:
-----
To list all your todo's:
node index.js list


add:
----
To add a new mission to your todo's:
node index.js add "[your new mission]"


remove:
-------
To remove an item from your todo's:
node index.js remove [1 based index of the wanted item]


reset:
------
To remove all items from your todo list:
node index.js reset


update:
-------
To update an existing item from your todo list with a new mission:
node index.js update [1 based index of the wanted item] "[your new mission]"
